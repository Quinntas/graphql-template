# GraphQL Temaplate

### This is a heavily opinionated template for graphql backend projects

### Features

-   [x] Latest packages
-   [x] Namespaces
-   [x] Clean architecture
- [x] DrizzleORM

### Prerequisites

-   [Node.js >= 20](https://nodejs.org/en/download)
-   [pnpm](https://pnpm.io/installation)

### Docker (MYSQL, REDIS, ADMINER)

```bash
docker-compose up -d
```

### Installing dependencies

```bash
pnpm install
```

### Running dev server

```bash
pnpm dev
```

### Running prod

```bash
pnpm build
```

```bash
pnpm start
```

### Running tests

```bash
pnpm test
```
